# DevOps Project 2 - Version Control with Git

## Overview

This project was completed as part of the DecodeLabs DevOps Industrial Training (Batch 2026). The objective is to understand the fundamentals of Git and GitHub by managing project files using version control.

## Objectives

* Create a local Git repository.
* Track project files using Git.
* Commit changes with meaningful commit messages.
* Push the repository to GitHub.
* Understand the basic Git workflow.
